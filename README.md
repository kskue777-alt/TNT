# TNT

## 앱 실행 방법
1. Flutter SDK 3.x와 Dart 3 이상이 설치되어 있는지 확인하세요.
2. 프로젝트 루트에서 Flutter 종속성을 설치합니다.
   ```bash
   cd app
   flutter pub get
   ```
3. 디버그 모드로 앱을 실행합니다.
   ```bash
   flutter run
   ```

## APK 받는 법
GitHub Actions 워크플로우가 실행되면 Debug APK가 빌드되어 아티팩트로 업로드됩니다. 워크플로우 실행 페이지의 **Artifacts** 섹션에서 APK를 다운로드할 수 있습니다.
